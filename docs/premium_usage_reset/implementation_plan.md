# プロプラン使用制限のリセット計画

テストを継続できるように、Proプランの月間使用制限（現在30回）をリセットします。

## 提案される変更

### データベース操作
直接データベース操作を行い、`PremiumUsage`テーブルのレコードを削除します。これにより、すべてのユーザーの使用回数が0にリセットされます。

#### 手順
1. `npx prisma db execute` を使用して、SQLで `DELETE FROM "PremiumUsage";` を実行する。
2. または、Prismaクライアントを使用した一時的なスクリプトで削除する。

## 確認計画

### 手動確認
1. アカウントページ（`/account`）またはヘッダーの使用量バッジを確認し、残り回数が「30回」にリセットされていることを確認する。
2. 実際に背景生成を1回行い、回数が正しくカウント（残り29回）されることを確認する。

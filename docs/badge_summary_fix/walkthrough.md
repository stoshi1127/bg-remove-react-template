# 修正内容の確認 (Walkthrough) - サマリーバッジの表示ロジック修正

フッターに表示されるサマリーバッジ（仕上、サイズ、背景）の表示内容を、現在の設定に合わせて正確に表示されるよう修正しました。

## 修正内容

### 1. 「仕上」ラベルの改善
- AI背景生成モードが選択されている場合、ラベルを **「仕上：AI画像合成」** と表示するようにしました。
- それ以外のモード（標準/高精度）も引き続き正しく表示されます。

### 2. 「背景」ラベルの改善
- **AI背景生成モード時**: ラベルを **「背景：AI生成」** と表示します。
- **カスタム画像アップロード時**: これまで「カスタム色」と表示されていた箇所を、正しく **「カスタム画像」** と表示するようにしました。
- **自然になじませる ON時**: 背景名の後ろに **「 (なじませる)」** が追記されるようになりました（例：`背景：レンガ (なじませる)`）。

## 動作確認のポイント

1. **背景をアップロード**: カスタム画像をアップロードし、バッジが「背景：カスタム画像」に変わることを確認してください。
2. **なじませるをON**: 「自然になじませる」をチェックし、バッジが「背景：カスタム画像 (なじませる)」に変わることを確認してください。
3. **AI生成を選択**: 「AIで背景を作る」を選択し、バッジが「仕上：AI画像合成」「背景：AI生成」になることを確認してください。

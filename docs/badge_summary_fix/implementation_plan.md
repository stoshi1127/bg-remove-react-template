# サマリーバッジの表示ロジック修正

フッターに表示されるサマリーバッジの内容が、現在の設定（AI背景生成、カスタム画像、なじませる機能）を正しく反映していない問題を修正します。

## 変更内容

### `BgRemover.tsx`

#### スティッキーフッター（Sticky CTA）内のバッジ表示

以下のようにラベルの算出ロジックを改善します。

- **仕上：**
  - `standard` -> `標準`
  - `pro_high_precision` -> `高精度`
  - `ai_generate` -> `AI背景生成`
- **背景：**
  - `selectedProcessingMode === 'ai_generate'` の場合 -> `AI生成`
  - `selectedTemplate` が画像データ（Data URL）の場合 -> `カスタム画像`
  - `selectedTemplate` がテンプレートリストにある場合 -> テンプレート名
  - それ以外（カラーコード等） -> `カスタム色`
  - `blendEnabled` が true の場合 -> 名前＋ ` (なじませる)` を付与
  - `selectedTemplate` が null の場合 -> `透過`

## 検証計画

### 手動確認
1. カスタム画像をアップロードし、バッジが「背景：カスタム画像」になることを確認。
2. 「自然になじませる」をONにし、バッジが「背景：... (なじませる)」になることを確認。
3. 「AIで背景を作る」を選択し、バッジが「仕上：AI背景生成」「背景：AI生成」になることを確認。

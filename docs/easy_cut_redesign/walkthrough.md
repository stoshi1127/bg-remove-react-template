# Stitchデザインの適用（トップページ上部）

## 概要
Stitchで生成された「イージーカット - アイコン・レイアウト改善版」のデザインを、トップページ（ヘッダーからアップロードエリアまで）に関連するコンポーネントへ反映しました。

## 実装した変更

### 1. `src/app/globals.css`
* Tailwind CSS（v4系）の変数設定として、Stitchデザインで使用されていた色（`primary`, `pro-orange`）を追加しました。

### 2. `src/components/HeaderClient.tsx`
* サイトのナビゲーションバー（ヘッダー）を新しいデザインに合わせて更新しました。
  * 背景にブラー効果（`backdrop-blur-md`）を追加しました。
  * ロゴとナビゲーションの文字色レイアウトを刷新しました。
  * ログイン・Pro版購入ボタンの色アサインとシャドウ等のスタイルをStitchのモックアップに沿って調整しました。

### 3. `src/app/page.tsx`
* ヒーローセクション（タイトルと説明文）のテキスト内容を更新しました。
* 新しいデザインに合わせて文字サイズ（`text-5xl`〜`text-6xl`）やマージンの微調整を行いました。
* 変更範囲はヒーローエリアのみとし、後半のセクション（使い方ガイドやFAQ等）は既存のまま残しています。

### 4. `src/components/UploadArea.tsx`
* 画像のドラッグ＆ドロップエリアのデザインを大幅に刷新しました。
* アニメーション効果（ホバー時にアイコンが拡大/回転する等）を追加し、リッチなUIへと変更しました。
* アイコンやテキストのスタイルを全体のトーンマナー（`primary` カラー基調）に合わせて統一しました。

## 実装した変更

* **切り抜きモード設定:** コンポーネントをStitchデザインに沿ったカード型のセレクトUIへと刷新しました。`Material Symbols`を利用したアイコン表示やProバッジを追加し、視認性を高めています。

* **背景カスタマイズ:** 背景無し、アップロード、カスタム色、テンプレート画像の選択UIを、横並びのモダンなカード・パレットUIに改善しました。
* **高度な設定:** 「ナチュラルブレンド」のチェックボックスを、iOSライクなトグルスイッチのアニメーションへと変更しました。

### 2. `src/components/BgRemover.tsx` (ファイルリストとアクションボタン)
* **ファイルリスト:** 各アイテムを角丸 (`rounded-3xl`) のカード型UIに改善しました。元の構造に大幅な変更は加えず、パディングやボーダーのクラスを調整しています。
* **右下フローティングアクションボタン:** モバイル・PC問わず、画面の右下へ「選択した画像の背景を透過する」ボタンをフロート表示するよう実装しました。
* 付随して実行ステータス状況（モード、サイズ、背景、枚数）を示す「チップ型の要約データ」もボタンの上部に表示し、機能ごとに直接スクロール可能なアンカーとして機能します。

# Stitchデザインの適用（料金プランエリア・フェーズ3）

## 概要
Stitchの「料金プランの比較」モックアップに基づき、トップページのProプラン訴求エリアを新しいカード形式のデザインへと刷新しました。

## 実装した変更

### 1. `src/components/PricingTable.tsx`
* 従来の `<table>` 要素を用いた機能比較表を撤廃し、`grid` を用いた「無料プラン」「Proプラン」が横並びになる2カラムのカードUIへと刷新しました。
* Proプランの特徴をチェックマークアイコンとともに箇条書きにするリスト形式に変更し、視認性を向上させました。
* Stitchでは青色がベースでしたが、当サイトのブランドカラー（`pro-orange`・`amber`系）を基調としたカラーリングへと統一しました。
* 料金や特典下部に購入ボタンを直接表示できるよう、Render Props（`renderProCta`）を受け取る仕組みを追加しました。

### 2. `src/components/ProCtaSection.tsx` および `src/components/PricingModal.tsx`
* トップページのCTAセクション（`ProCtaSection`）を調整し、価格とボタンが独立していた右側カラムを廃止し、新しい `PricingTable` を中央に配置する1カラムのすっきりしたデザインへ修正しました。
* 各種購入ボタン（`GuestProPurchase`）を `renderProCta` 経由でカード内に注入するように連携させました。
* ヘッダー等から呼び出される「Proを購入」モーダル（`PricingModal`）も、新しい `PricingTable` のサイズ拡張に合わせて、モーダルの最大幅（`max-w-3xl` → `max-w-5xl`）を広げ、自然に見えるように調整しました。

## 完了の確認
* [x] **README確認**: 本質的な料金・仕様の変更はなく、UI上の見せ方の変更のみであるため、README.md等の更新は不要です。
